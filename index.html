11. Conceptos de POO

Es un paradigma de programación asi como otros, que se basa en, como su nombre lo indica, crear objetos
Tiene algunas caracteristicas como:
--Abstracción: Intentar reducir el objeto a sus componentes basicos, por ejemplo, si tenemos un perro, no decimos tamaño de la cabeza, color de la cola, etc
--Modularidad: Resolver un problema grande diviviendolo por partes
--Encapsulamiento: Hacer que todos los datos sean privados, encapsular los datos
--Polimorfismo: Ver como un objeto se comporta diferente ante un metodo, por ejemplo, si tengo el metodo caminar, un perro va a caminar diferente que un pajaro por las propiedades que tiene
--Herencia: Sirve para agregarle mas propiedades o metodos a una clase, heredando cosas de la clase principal
Ejemplo de Herencia:

    class Perro extends Animal{
        constructor(especie, edad, color, raza){
            super(especie, edad, color);
            this.raza = raza;
        }
    }

const perro = new Perro("perro", 7, "negro", "Pastor aleman");

 Y ahora puede tener los metodos de la clase Animal, y los parametros que puso en super

--Metodos estaticos: Son los metodos de una clase que se pueden utilizar antes de crear un objeto o que no necesitan que el objeto se haya creado para poder usarse
Ejemplo:
    class Perro extends Animal{
        constructor(especie, edad, color, raza){
            super(especie, edad, color);
            this.raza = raza;
       }
      static ladrar( ){
          alert("Wof");
      }
    }

    Perro.ladrar( );
 Como podemos ver, la clase Perro, puede usar el metodo static sin haber creado el objeto por que no utiliza ningun parametro


--Metodo Get y Set:
Get se utiliza para traer el valor de un parametro y set para asignarlo
Ejemplo:

    class Perro extends Animal{
        constructor(especie, edad, color, raza){
            super(especie, edad, color);
            this.raza = raza;
       }

      set setRaza(newRaza){
          this.raza = newRaza;
      }

      get getRaza( ){
          return this.raza;
      }
    }

    perro.setRaza = "Pedro";
    document.write(perro.getRaza);

Esto imprime la raza pedro por que la modificamos con set y trajimos el valor con Get


12. Metodos de cadena

Son metodos que se pueden utilizar para modificar las cadenas de texto.

----Concat( )----
Junta dos o mas cadenas y las retorna.
Ejemplo:
      let cadena = "cadena 1";
      let cadena2 = "cadena 2";

      resultado = cadena.concat(cadena2);

      document.write(resultado);



----startsWith( ) y endsWith( )----
Si una cadena empieza o termina con los caracteres de otra cadena devuelve true, sino false
Ejemplo:
      let cadena = "cadena de prueba";
      let cadena2 = "cadena";

      resultado = cadena.startsWith(cadena2);

      document.write(resultado);
Esto devuelve True en pantalla, por que si empieza con esa cadena, con endsWith devuelve false, por que no temina asi.



----includes----
Si una cadena se encuentra (en cualquier parte) dentro de otra cadena devuelve true, sino false


----indexOf y lastIndexOF----
Devuelve el indice del primer o ultimo caracter de la cadena, si no existe, devuelve -1
Ejemplo:
      let cadena = "cadena de prueba si";
      let cadena2 = "prueba";

      resultado = cadena.indexOf(cadena2);

      document.write(resultado);
Esto devuelve 11 en pantalla, por que el primer caracter de la cadena, la "p", está en la posicion 11
si fuera lastIndexOf( ) sería 16 por que es el indice del ultimo caracter de la cadena



----padStart( ) y padEnd( )----
Rellena la cadena al principio o final con los caracteres deseados.
Ejemplo:
      let cadena = "abc";
      let cadena2 = "";

      resultado = cadena.padStart(10, "12");

      document.write(resultado);

Esto devuelve 1212121abc en pantalla, se agregan los necesesarios "12" para completar los 10 caracteres que se definieron, aunque falte uno se pone para llegar a 10, si fuera el padEnd: abc1212121



----repeat( )----
Devuelve la misma cadena pero repetida la cantidad de veces que deseamos
Ejemplo: abcabcabc si escribimos al ejemplo de arriba resultado = cadena.repeat(3);



----split( )----
Divide la cadena en un array, desde donde nosotros le indicamos
Ejemplo:
      let cadena = "Hola como estas";
      resultado = cadena.split("como");

      document.write(resultado[0]);

Esto devuelve Hola en pantalla, por que separamos desde donde nosotros le indicamos
el como se elimina
en resultado[1] está el estas
y si pusieramos solo un espacio, separa esa cadena en tres partes por los tres espacios: [Hola,como,estas]



----substring( )-----
Nos retorna un pedazo de la cadena empezando y terminando desde donde nosotros deseamos
Ejemplo:
      let cadena = "ABCDEFG";
      resultado = cadena.substring(2,4);

      document.write(resultado);

Esto devuelve CD en pantalla, ya que toma la posicion 2 y 3 porque OJO toma el indice del inicio pero no el del final



----toUpperCase( ) y toLowerCase( )----
 Pasa la cadena de minuscula a mayuscula y viceversa



----toString( )----
Pasa a String



----trim( ), trimStart( ) y trimEnd( )----
Quita los espacios al inicio y al final de una cadena, solo los espacios del inicio o solo los del final respectivamente.


13. Metodos de Arrays


----Transformadores----

--Pop( ) : Elimina el ultimo elemento de un array y lo devuelve
Ejemplo:
      let nombres = ["pedro", "maria", "jorge"];
      let resultado = nombres.pop();

      document.write(nombres);

 Aqui muestra jorge por que fue el que eliminó por ser el ultimo elemento del array



--shift( ) : Elimina el primer elemento de un array y lo devuelve, asi como el anterior.



--push( ) : Agrega uno mas elementos al final del array y devuelve la cantidad de elementos, o la posicion del ultimo elemento que es lo mismo. let resultado = nombres.pop("Elemento n");



--unshift( ) : Agrega uno o mas elementos al inicio del array y devuelve la cantidad de elementos.



--reverse( ) : Invierte el orden de los elementos del array.



--sort( ) : Ordena los elementos de un array localmente y lo devuelve ordenado (si hay numeros los ordena de menor a mayor o si son palabras, alfabeticamente)



--splice( ) : Cambia el contenido de un array eliminando elementos existentes y/o agregando nuevos elementos. numeros.splice(2, 1, "Juan") con esto, decimos desde que posicion vamos a empezar a trabajar, desde que elementos vamos a eliminar, y si vamos a agregar elementos los especificamos ahi.



----Transformadores----

--join( ) : Une todos los elementos de un array en una cadena y lo devuelve, es como el toString( ), pero mejor, por que deja poner como lo vamos a separar al integrarlo a la cadena
Ejemplo:
      let nombres = ["pedro", "maria", "jorge"];
      let resultado = nombres.join(" - ");

      document.write(resultado);

 Esto muestra: pedro - maria - jorge como una cadena de texto



--slice( ) : devuelve una parte de nuestro array, desde donde y hasta donde nosotros le indicamos,
let resultado = numeros.slice(1, 5) el ultimo numero no está incluido



--Los metodos de cadena que ya vimos como: toString( ), indexOf( ) que devuelve en que posicion del array está el elemento que le indicamos, includes( ) que nos dice true o false si lo incluye



----De repetición----

--filter( ) : crea un nuevo array con todos los elementos que cumplan la condicion
filter( ) trabaja con una funcion y en cada vuelta, el parametro va a ser igual a cada elemento del array y podemos ponerle una condicion Ejemplo:
      let nombres = ["abecedario", "manzana", "pedro", "dedo", "bob", "pedro"];
      let resultado = nombres.filter(let numero => numero.length > 4);

      document.write(resultado);

 Esto muestra en pantalla abecedario, manzana, pedro, pedro por que los otros dos no cumplen con la condicion de tener mas de cuatro letras

--Elemento.classList.replace()
Ponemos dentro del parentesis, se paradas por comas, primero la clase a reemplazar, y segundo la clase con la que la vamos a reemplazar, si todo esta bien y reemplaza nos devuelve true, si la primer clase no existiera o no se reemplazara nos diera error
