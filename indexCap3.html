JavaScript Nivel 3





















































1. Generalidades

-Para asignar un valor es con =
-Para decir que es 'igual a' es ==
-Para imprimir algo en pantalla es document.write("mensaje")
-Para mostrar un cuadrito arriba con información es alert("Mensaje")
-Para mostrar el mismo cuadrito del alert pero para que ingrese información es prompt("Escribir mensaje:")
-Para cambiar el valor de String a Int es con variable = parseInt(variable); o con let num = Number(str);
-Con typeof variable muestra el tipo de variable si es number, string o cualquier otro
-Para asignar cuantos decimales tiene un Float es con numObj.toFixed([digitos])
-Es keysensitive, significa que diferencia entre mayuscula y minuscula


2. Arrays

Es como una variable pero podemos guardar muchas cosas en el
Por ejemplo:
      let frutas = ["pera", "manzana", "fresa"];

Podemos mostrar todos los elementos, o individuales asi:
document.write(frutas[1]);
 Que imprime: manzana por que empieza desde el cero


Tambien existen Arrays Asociativos con los que identificamos los nombres de los elementos
Ejemplo:
let Computadora = {
      nombre: "TiBOPC",
      procesador: "intel Core i7",
      RAM: "16 GB",
      Espacio: "1TB",
      Lectora: "DVD"
};

document.write(Computadora["nombre"])
 Que imprime: TiBOPC por que identificamos que "nombre" tenia ese valor
